# Dual Knob Playground 동작 설명서

## 페이지 진입

- 홈: `/pages/index.html`
- 노브 드로잉: `/pages/knob-drawing.html`
- 미로 탈출게임: `/pages/maze.html`
- 금고 따기 퍼즐: `/pages/safe-crack.html`
- 리듬 노브 게임: `/pages/rhythm-knob.html`

## 노브 드로잉 규칙

- 좌클릭: `색칠 ↔ 이동` 토글
- 우클릭: `지우개(화이트) ↔ 이동` 토글
- 좌+우 동시 클릭 또는 `P`: 색상 팔레트 열기/닫기
- 색 선택 시: 팝업 닫기 + 이동 모드 복귀

## 노브 드로잉 캔버스 크기

- `10x10`, `15x15`, `30x30`, `50x50`, `100x100`
- 선택 시 모눈 칸 수 기준으로 보드가 생성됨

## 미로 탈출게임 규칙

- 목표: 파란 시작점에서 초록 도착점까지 이동
- 좌클릭 또는 `Q`: 이동 토글 ON/OFF
- 우클릭 또는 `E`: 부스트 토글 ON/OFF
- 좌+우 동시 클릭 또는 `P`: 미로 크기 선택 팝업
- 방향 이동: `WASD` 또는 방향키
- 부스트 ON: 한 번 입력에 최대 2칸 이동 시도

## 금고 따기 퍼즐 규칙

- 좌 노브 회전 → 다이얼 A(0~99) 조작
- 우 노브 회전 → 다이얼 B(0~99) 조작
- 좌/우 노브 클릭 → 현재 핀 해제 시도
- 두 다이얼이 동시에 정답 허용오차 이내일 때만 핀 해제 성공
- 실패 시 시간 -3초 페널티
- 단계별 제한시간: `40, 40, 60, 60, 80, 80, 100, 100, 120, 120초`
- 난이도 차이:
  - 초급: 핀 3개 시작 / 허용오차 4
  - 중급: 핀 4개 시작 / 허용오차 3
  - 고급: 핀 5개 시작 / 허용오차 2
- 단계가 진행될수록 핀이 늘고 허용오차가 좁아짐
- 10단계 모두 클리어 시 누적 클리어 시간이 기록(localStorage)

## 리듬 노브 게임 규칙

- 좌/우 버튼: Tap 노트 입력
- 좌 노브 회전(Y): LEFT 레인 Spin/Flick 노트 입력
- 우 노브 회전(X): RIGHT 레인 Spin/Flick 노트 입력
- 중클릭: 일시정지/재개
- 판정: `Perfect`, `Great`, `Good`, `Miss`
- 점수는 판정 + 콤보 배수로 누적
- 결과 화면에서 점수/정확도/최대 콤보 확인 가능
- 난이도별 최고 점수와 정확도가 localStorage에 저장됨
