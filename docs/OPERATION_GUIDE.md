# Dual Knob Playground 동작 설명서

## 페이지 진입

- 홈: `/pages/index.html`
- 노브 드로잉: `/pages/knob-drawing.html`
- 미로 탈출게임: `/pages/maze.html`

## 노브 드로잉 규칙

- 좌클릭: `색칠 ↔ 이동` 토글
- 우클릭: `지우개(화이트) ↔ 이동` 토글
- 좌+우 동시 클릭 또는 `P`: 색상 팔레트 열기/닫기
- 색 선택 시: 팝업 닫기 + 이동 모드 복귀

## 노브 드로잉 캔버스 크기

- `10x10`, `15x15`, `30x30`, `50x50`, `100x100`
- 선택 시 모눈 칸 수 기준으로 보드가 생성됨

## 미로 탈출게임 규칙

- 목표: 파란 시작점에서 초록 도착점까지 이동
- 좌클릭 또는 `Q`: 이동 토글 ON/OFF
- 우클릭 또는 `E`: 부스트 토글 ON/OFF
- 좌+우 동시 클릭 또는 `P`: 미로 크기 선택 팝업
- 방향 이동: `WASD` 또는 방향키
- 부스트 ON: 한 번 입력에 최대 2칸 이동 시도
