# Dual Knob Drawing Tool 기획서

## 1. 목표

두 개의 노브를 이용해 그림을 그리는 웹 기반 드로잉 툴을 만든다.

- 왼쪽 노브: Y축 이동
- 오른쪽 노브: X축 이동
- 왼쪽 노브 푸시: 펜 ON/OFF
- 오른쪽 노브 푸시: 지우개 또는 색상 변경
- 두 노브 동시 회전: 대각선 이동

## 2. 개발 방식

### 1차 버전

Arduino Leonardo 또는 RP2040이 USB 마우스처럼 동작한다.

- RP2040 상세 실행 태스크: `docs/RP2040_TASKS.md`

```text
노브 회전 -> 마우스 이동
버튼 클릭 -> 마우스 클릭
브라우저 캔버스 -> 그림 그리기
```
